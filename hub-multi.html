💡 밀크방 PDF를 CSV로 변환하는 방법

새 대화 시작하기 →

2. PDF 업로드 후 아래 프롬프트 복사해서 입력:
이 PDF를 구역별로 CSV 파일로 만들어줘. 형식: 순번,도로명주소,세부주소,참고사항,품목 구역 인식 방법: - PDF 상단에 [숫자]구역명 형식으로 표시됨 (예: [0917]중앙로, [2122]아라동, [5006]도남동) - 각 구역은 이 표시로 구분됨 - 각 구역별로 별도의 CSV 파일 생성 중요 규칙: 1. 각 배송지는 빈 줄로 구분되어 있음 2. 한 배송지에 여러 품목이 있으면 한 행으로 합치기 ⚠️ 주의: 품목이 다음 줄이나 다음 페이지로 이어질 수 있음 예: "닥터칸 3" 다음 줄에 "헬저지방 소 3" → 같은 고객! 3. 품목: "건국 소 1 / 오메가수 소 2" 형식으로 "/" 구분 4. 참고사항: 주소 바로 아래 괄호 안 내용만 포함 (예: (주차장), (#1234#), (문앞비닐)) 5. 제외 대상: - "주문(휴X)" 컬럼의 정보 (예: 월2목1, 월목) - 품명/수량 옆의 배송 주기 정보 특별 처리 필요: - PDF가 2단 레이아웃인 경우: 왼쪽 열 전체 → 오른쪽 열 전체 순서로 읽기 - 페이지 넘김: 순번이 이어지면 같은 배송지로 합치기 - 순번만 있고 주소 없으면 = 이전 배송지의 추가 품목 구역별 파일명: 중앙로.csv, 아라동.csv, 도남동.csv 등
3. 생성된 CSV 파일들을 다운로드
4. 여기에 한 번에 업로드!
